RNA-seq gene expression and pathway analysis - supplemental material

Katy Maher, Helen Hipperson, Steve Paterson, Bert Overduin, Matthew Gemmell, Xuan Liu

2022-11-08

Chapter 1 Introduction

This bookdown is supplemental to the main one. It contains some additional code and examples of tools that may prove useful in addition to the core RNA-Seq pipeline.

The sections in this supplement will cover:

  • Introduction to R
  • Making a count table from HTSeq output files
  • Further analysis with DESeq2

Ensure you have the rnaseq environment activated for using all of these materials. As a reminder, the command to do this (if you are not still in the R session) is:

. usernaseq
