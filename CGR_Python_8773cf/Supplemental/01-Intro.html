1 Introduction



These supplementary materials are focused on working with other types of data and generating plots with Python. We will work with comma delimited data tables and create a bar chart and scatter plot.

These material serves as an introduction to two very powerful Python libraries:

Pandas Matplotlib

This bookdown will also expand on the use of loops, reading and writing files, subsetting data, and applying functions that you have already practiced.
